Install
------------------------------------------------------------------------------

``gh_org_gov`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install gh-org-gov

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade gh-org-gov
